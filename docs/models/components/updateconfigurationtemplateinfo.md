# UpdateConfigurationTemplateInfo Object
### Description
The `ConfigurationTemplate` object is used to represent an update to a configuration template.

### Usage Example
Use in the `PUT Configuration Templates` endpoint.

## Example Usage

```typescript
import { UpdateConfigurationTemplateInfo } from "opal-mcp/models/components";

let value: UpdateConfigurationTemplateInfo = {
  configurationTemplateId: "7c86c85d-0651-43e2-a748-d69d658418e8",
  name: "Prod AWS Template",
  adminOwnerId: "7c86c85d-0651-43e2-a748-d69d658418e8",
  visibility: {
    visibility: "GLOBAL",
  },
  linkedAuditMessageChannelIds: [
    "37cb7e41-12ba-46da-92ff-030abe0450b1",
```
